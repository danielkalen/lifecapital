  2. Full Height Intro


    As you may have noticed, the hero (intro) section is always at 100% the height of the browser. The reason for this is because I wanted to add as much focus as possible to the heading of the page and make sure that the most important things of the page will be surely noticed.


  3. Heading Text


    Heading is always centered and is the largest because it requires the most attention.

  4. Background Darkness

This also makes the background subtle so it wouldn't distract from the important content but it'll still anchor the tone/mood for the heading on a psychological level (i.e. 'Rleax...').

  5. Scroll Indicator


    Since the intro banner is at full height, some users might not know that they need to scroll. Therefore a subtle indicator was added.


  6. White Space

    You may have noticed that across the website there is plenty of whitespace (or empty spaces) that may not seem reasonable. But having plenty of whitespace is one of the most important parts of a successful design. The reason for this is because human beings cannot process too much information at a time, making it necessary to decrease cognitive load as much as possible across the website. See Dropbox's website for example.

    Psychological Evidence:

    Source 1
    Source 2

  7. Scrolling Animations

    People skim through websites quickly. Because of this, they skip thourgh important messages that you want to deliver to them. I came up with these scrolling animations as a solution to this skipping problem since no matter at which speed the user scrolls, the animations will catch his attention and will most likely make him scroll back to see the source of the animation thus making him read the important content. Additionally, these animations add a great deal of beauty to the website.

  8. Only the important

    Unlike On Bond Street's website, only the important content is visible to the user as first sight. If he wants to see more, he can hover or click on each 'opprotunity' which opens up a beautiful modal popup (unlike On Bond Street which requires loading a seperate page to view the details of the story). This removes clutter and cognitive load.

  9. More Scrolling animations

    Again the illustration fills with the user's scroll to catch the user's attention and add to the aesthetics of the website.

  10. Emphasis on 'steps'

    Each step enters with the user's scroll one after the other, visually expressing the 'steps' (1,2,3) and making sure to catch the user's attention.

  11. Advanced Form

    Unlike other forms, this form is much more interactive since all calculations are done live and the form processing occurs without having the page refresh and waste the user's valuable time. Also, between each step there are advanced animations to keep the user interested while waiting for the form to process.

  12. Interactive Buttons

    Button remains disabled until the user completes all of the fields. Once all fields are completed, the button turns blue to indicate the next step and motivate the user to press continue.

  13. Subtle Footer

    Usually on other websites the last section of the website contains a 'footer' with various links and information that distracts the user from the above content. I've made sure to keep only the important stuff in a subtle format so that the user doesn't get distracted from the main call to action (the above form).
